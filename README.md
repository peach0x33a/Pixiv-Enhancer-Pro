## 这是什么? / What this?
这是一个Pixiv功能增强脚本，目前主要聚焦在改善下载体验

本人是第一次写这类用户脚本，同时大量使用（95%）了 Google Gemini 来辅助，如有问题请多多包涵，欢迎pullrequests

极有可能荒废的项目。
